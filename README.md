# Welcome to Computational Tools for the Modern Mathematician: Git, Github & PostgreSQL

You are officially welcome to computational tools for the modern mathematician: Git, Github & PostgreSQL.  

## Prerequisites
To successfully complete this project, the instructor assumes that  
- You have a little experience with programming
- You know how to work with text editors
- You have a little experience working with terminals or shells.

Do no panic if you do not have any of the above skills, hopefully we land the plane safely :).
